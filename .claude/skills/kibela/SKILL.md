# Kibela 記事検索

## 実行指示

引数で指定されたキーワードで Kibela を検索し、関連記事をサマリー表示してください。

### 手順

1. **Kibela MCP で検索**する（引数をそのままクエリとして使用）

2. **上位 3〜5 件の記事を取得**して内容を確認する

3. **以下の形式でサマリーを表示**する

```
## Kibela 検索結果: "{キーワード}"

### 1. [記事タイトル]
URL: {記事URL}

**要点**:
- （記事の重要ポイントを箇条書き 3〜5 点）

**実装への示唆**:
（今の開発にどう使えるか一言）

---

### 2. [記事タイトル]
...
```

4. 検索結果が 0 件の場合は「"{キーワード}" に関する記事は見つかりませんでした。別のキーワードを試してください。」と返す

### 注意

- キーワードが指定されていない場合は「検索キーワードを引数で指定してください（例: /kibela 認証フロー）」と返す
- 記事が長い場合は冒頭と見出し構造を中心に要約する
