# 日報差分コピー

## 実行指示

以下の手順で本日の日報の差分をクリップボードにコピーしてください。

### 1. 差分の取得

`$HOME/nippo` リポジトリで、日報ファイルの差分を取得する。

```bash
cd $HOME/nippo
```

以下の優先順位で差分を取得する：

1. **未コミットの変更がある場合**: `git diff -- .` の結果を使用
2. **ステージ済みの変更がある場合**: `git diff --staged -- .` の結果を使用
3. **変更がない場合（clean）**: 直近のコミットとの差分を使用 `git diff HEAD~1 -- .`

### 2. クリップボードへコピー

取得した差分を `pbcopy` でクリップボードにコピーする。

### 3. 結果の報告

コピーした差分の概要（追加行数・削除行数）をユーザーに報告する。
差分がない場合はその旨を報告する。
