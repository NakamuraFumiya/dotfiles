# Jira チケット確認

## 実行指示

引数で指定されたチケット ID（例: `PROJ-123`）の内容を Jira MCP で取得し、以下の形式で整理して表示してください。

### 取得する情報

- **タイトル**
- **ステータス**（未着手 / 進行中 / 完了 など）
- **優先度**
- **担当者**
- **説明**（要約して箇条書きに整理）
- **受け入れ条件**（Acceptance Criteria があれば抜粋）
- **ラベル / コンポーネント**

### 出力形式

```
## [チケット ID] タイトル

**ステータス**: xxx  **優先度**: xxx  **担当者**: xxx

### 概要
（説明を簡潔にまとめた箇条書き）

### 受け入れ条件
（あれば箇条書き）

### メモ
（実装上の注意点や関連チケットがあれば）
```

チケット ID が指定されていない場合は「チケット ID を引数で指定してください（例: /ticket PROJ-123）」と返してください。
